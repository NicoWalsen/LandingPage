# Landing Page BarbiGo - Carnes Premium

Este proyecto es una landing page para una tienda en línea de carnes premium importadas de EE.UU., con énfasis en cortes High End Choice y Wagyu, así como productos BBQ especiales.

## Características

- Diseño totalmente responsivo para todos los dispositivos
- Secciones principales: hero, nosotros, categorías, productos, testimonios y contacto
- Filtrado de productos por categorías
- Funcionalidad de carrito de compras (simulación)
- Formularios de contacto y newsletter
- Animaciones de scroll y transiciones
- Menú de navegación móvil con animaciones

## Tecnologías utilizadas

- HTML5
- CSS3 con variables y flexbox/grid para layouts
- JavaScript (ES6+)
- Intersection Observer API para efectos de scroll
- Font Awesome para iconos
- Google Fonts para tipografías

## Estructura de archivos

```
landing-page-barbigo/
├── index.html          # Estructura principal HTML
├── styles.css          # Estilos CSS
├── script.js           # Funcionalidad JavaScript
└── Imágenes productos/ # Carpeta con imágenes de productos
```

## Cómo usar

1. Clona este repositorio o descarga los archivos
2. Abre el archivo `index.html` en tu navegador
3. Para desarrollo, puedes editar los archivos con cualquier editor de código

## Personalización

- Colores: Se utilizan variables CSS en el archivo `styles.css` que pueden modificarse fácilmente
- Fuentes: Se utilizan 'Playfair Display' para títulos y 'Montserrat' para texto
- Imágenes: Puedes reemplazar las imágenes en la carpeta 'Imágenes productos'
